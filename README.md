# g0v0-client-versions

Version hashes for g0v0-server clients. These files are auto-generated by the CI workflow from the [script](https://github.com/GooGuTeam/g0v0-client-versions/tree/script) branch.

- [version_list.json](version_list.json) — default client version hashes
- [community_versions/](community_versions/) — community client version hashes
